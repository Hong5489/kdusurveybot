# KDU Survey Auto Complete Bot
## How to install
```
cd kdusurveybot
pip install -r requirements.txt
```

## How to use
```
# Normal Mode
python bot.py -u 0120000 -p 0000

# All same rating mode
python bot.py -u 0120000 -p 0000 -all 5 # All rating 5
```
